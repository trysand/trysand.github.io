FAQ s

    Sand is a self hosted help desk tool , where you have to pay a minimal fees in order to install source code on your sever. Pay once and get free updates for lifetime

  2. What are the requirements to install Sand ?

    You need to have a web server up and running with Php, Mysql and Apache / Nginx , complete requirements for specific Php versions will be shared after launch.

  3. How painful or easy it will be install to Sand ?

    It is as easy as setting up a new email with Gmail or Yahoo. Also there will be complete user and developer guide to get started with Sand.

  4. What will be the maximum price for Sand ?

    It will never exceed 100 USD , the fee is for lifetime with includes all upcoming updates with new features and bugs fixing.

  5. Is there any scope for developers ?

    Ofcourse, Sand is built on top of REST API, which allows you and your team to write modules in any programming language.

    Upcoming developer guide will help in understanding Sand structure and build modules on top of it.

  6. At last

    Sand is built on top of Laravel ( a php framework ) and AngularJs, it makes use of queues and events to make end user experience better.
